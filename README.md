## 🚀 Quick Start


1. **Clone & Install**
   ```bash
   git clone https://github.com/renocrypt/synthrus-chem.git
   cd synthrus-chem
   npm install
   ```

2. **Start Development Server**

```bash
npm run start
```

3. **Build for Production**

```bash
npm run build
```

4. **Deploy to GitHub Pages**
```bash
npm run deploy
```
